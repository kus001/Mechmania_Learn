    To follow this course, you need a personal Linux or Windows computer (Linux or [WSL](https://learn.microsoft.com/en-us/windows/wsl/) is recommended). You will not be able to install the Arduino IDE on a school-managed Chromebooks. Instead, you should see the [Arduino Cloud editor](https://app.arduino.cc) to code in the browser.
:::
